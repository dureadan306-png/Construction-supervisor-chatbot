# Construction-supervisor-chatbot
A digital profile of my father ,built as an interactive python chatbot.
An interactive conversational chatbot that preserves my father's life story, professional journey, and values.

## 📌 Project Description
This project is a Python-based digital persona built to simulate an interactive conversation about my father, Muhammad Arshad. It captures his professional identity as a construction supervisor and his personal character, highlighting his dedication, patience, and kind nature.

## 🚀 Key Features
* **Biographical Data:** Learn about his name, education, and career.
* **Professional Insights:** Details on his work in construction site management and labor supervision.
* **Personal Anecdotes:** Insights into his personality, daily routines, family values, and life philosophy.
* **Interactive Interface:** A persistent command-line loop that responds to natural language queries.

## 🛠 Tech Stack
* **Language:** Python
* **Logic:** Keyword-based Heuristic Matching
* **Architecture:** Command-Line Interface (CLI)
